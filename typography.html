Наш шрифт в бренде

Типографика является важным компонентом визуального стиля. Наш шрифт Artos Sans имеет яркий голос, одновременно дружелюбный и авторитетный, а также обеспечивает единообразие во всех точках контакта и формах общения.

Айдентика бренда уделяет большое внимание современному типографскому выражению, но также гарантирует, что в результате не пострадает ясность коммуникации. Artos Sans и его обработка обеспечивают визуальную согласованность всей нашей коммуникации и помогают повышать узнаваемость и укреплять бренд.

Artos Sans доступен в четырех вариантах начертания: Light, Regular, Medium и Black.

Каждый из четырех весов имеет свою собственную функциональность и применение. Это означает, что при построении шрифта, были учтены все заголовки, основной текст и вспомогательные типографские знаки и символы.

С этими доступными параметрами легко общаться четко во всех типах приложений, от больших сообщений до небольших, более информативных текстов. Очевидный контраст между весами обеспечивает четкое определение между ними и означает, что при необходимости легко достичь типографской иерархии.

Заголовки

Artos Sans Medium предназначен для заголовков. Его также можно использовать для выделения небольших текстов, но в основном он используется для выделения важных сообщений.

Основной шрифт

Artos Sans Light используется для всех областей основного текста и мелкого текста. Это самый эффективный из четырех весов для удобочитаемости, и поэтому его лучше всего использовать, когда вы пытаетесь передать информацию четко и практично.

Вспомогательные шрифты

Artos Sans Regular используется для обеспечения типографского контраста и часто сочетается с Light или Medium. Он редко используется изолированно. Чаще всего он используется для областей текста, которые выделяются в большой типографской композиции.

Artos Sans Black в основном применяется для обозначения цифр, номеров и цен. Этот шрифт всегда оформляется с уменьшенными leading и letter-spacing.

Шрифт для Web

Фирменный шрифт Artos Sans подключается к цифровым материалам в формате WOFF. Для наилучшего его отображения необходимо правильно выставить значения расстояния между буквами и строками.

Формат

Для вебсайта и цифровых материалов всегда применяется формат WOFF2.


@font-face {
  font-family: 'Artos-Light';
  src: url(Artos-Light.woff2) format('woff2');
}
@font-face {
  font-family: 'Artos-Regular';
  src: url(Artos-Regular.woff2) format('woff2');
}
@font-face {
  font-family: 'Artos-Medium';
  src: url(Artos-Medium.woff2) format('woff2');
}
@font-face {
  font-family: 'Artos-Black';
  src: url(Artos-Black.woff2) format('woff2');
}

CSS letter-spacing Property

Для нашего шрифта очень важно подбирать правильно размер параметра "letter-spacing". Значение меняется в зависимости от его размера.


h1 {
  font-family: "Artos-Medium", Helvetica, sans-serif;
  font-weight: 500;
  font-size: 3rem;
  letter-spacing: -0.1rem;
}
h2 {
  font-family: "Artos-Medium", Helvetica, sans-serif;
  font-weight: 500;
  font-size: 2.25rem;
  letter-spacing: -0.075rem;
}
h3 {
  font-family: "Artos-Medium", Helvetica, sans-serif;
  font-weight: 500;
  font-size: 1.5rem;
  letter-spacing: -0.035rem;
}
h4 {
  font-family: "Artos-Black", Helvetica, sans-serif;
  font-weight: 900;
  font-size: 1.5rem;
  letter-spacing: -0.085rem;
}
p.major {
  font-family: "Artos-Regular", Helvetica, sans-serif;
  font-weight: 400;
  font-size: 1.2rem;
  letter-spacing: -0.035rem;
}
p {
  font-family: "Artos-Regular", Helvetica, sans-serif;
  font-weight: 400;
  font-size: 1rem;
  letter-spacing: 0 !important;
}
p.small {
  font-family: "Artos-Light", Helvetica, sans-serif;
  font-weight: 300;
  font-size: 0.8rem;
  letter-spacing: 0 !important;
}

Применение

Мы используем правило «трех» в качестве отправной точки, которая определяет многие наши дизайнерские решения.

Однако не обязательно всегда использовать три столбца. Сетка, основанная на числах, кратных 3 (6, 9, 12 и т. д.), может использоваться для обеспечения гибкости и создания различных макетов.

Обратите внимание, что правило «трех» является всеобъемлющей концепцией для создания единства всей системы дизайна и существует для того, чтобы обеспечить первоначальное вдохновение для макетов дизайна. Это правило призвано помогать, а не мешать, поэтому его следует использовать как источник вдохновения, а не ограничения.

Заголовки

Не стремитесь чрезмерно украшать пространство. Почувствуйте уверенность в нашем выражении. Негативное пространство — элемент дизайна. Не бойтесь давать контенту пространство для дыхания.

Создайте визуальную иерархию, особенно с помощью типографского контраста. Ищите подходящие моменты, чтобы быть выразительными. И не стесняйтесь позволить типографике быть выразительной.

Основной текст

С основным текстом и меньшим текстом возникает множество различных артикуляций и дополнений.

Суть в том, чтобы всегда использовать Artos Regular и быть практичным с точки зрения того, как это представлено. Вертикальные столбцы помогают нам повторить визуальный типографский язык бренда и, опять же, основаны на правиле «трех». Они могут быть жесткими по своей композиции или более выразительными, создавая динамичные композиции.

Старайтесь избегать предложений с шириной более 60 символов при настройке блока текста. Подписи должны помочь достичь типографского напряжения в любой композиции. Малые заголовки должны следовать за сеткой или разделением столбцов.

Как в печатных, так и в цифровых материалах взаимодействия наши размеры точек и пикселей следуют одной и той же системе.

Мы используем числа, кратные 3 и 6, чтобы определить наши размеры. Это гарантирует, что, несмотря на то, что у нас есть рекомендуемый набор часто используемых размеров, также есть возможность добавить больше, если есть особые потребности.

При больших размерах рекомендуется уменьшить leading (расстояние между строками) чтобы добиться правильного визуального выражения. А tracking (расстояние между буквами) остается неизменным.

При меньшем размере leading часто пропорционально увеличивается, чтобы максимизировать разборчивость и создать более подходящее выражение для основного текста.
